#  Diabetes Type Identifier using CIT


This project is a Machine Learning web application that identifies the diabetes category:

* **No Diabetes**
* **Prediabetes**
* **Diabetes**

The system uses a **Conditional Inference Tree (CIT)** classifier to predict diabetes type based on medical and health parameters such as:

* HbA1c
* BMI
* Age
* Glucose Levels
* Other clinical features



##  Model Used

* **Conditional Inference Tree (CIT)**
* Multiclass Classification
* Trained on a diabetes dataset


##  Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas
* HTML/CSS



##  How to Run

### 1️⃣ Clone the Repository


git clone https://github.com/krupa216/Diabetes_type_identifier.git


### 2️⃣ Navigate to Project Folder


cd Diabetes_type_identifier


### 3️⃣ Install Dependencies


pip install -r requirements.txt

### 4️⃣ Run the Application


python app.py


### 5️⃣ Open in Browser


http://127.0.0.1:5000/
